## 1. Getting Started

When you open the Quill Editor, you'll see a familiar word-processor-like interface:

- **Toolbar**: The row of buttons at the top. This is where you'll find all formatting options.
- **Editor Canvas**: The large white area below the toolbar. This is where you type and edit your document.

**Your First Step**: Click anywhere in the Editor Canvas and start typing. It's that simple!

## 2. Typography & Text Formatting

This section covers how to control the appearance of your text: fonts, sizes, styles, and colors.

### 2.1 Choosing a Font

The editor provides **7 carefully selected fonts** to suit different content types:

| Font Name | Best For | Example Use Case |
|-----------|----------|------------------|
| **Sans Serif** (Default) | General content | Everyday documents |
| **Roboto** | Modern, clean look | Tech reports, websites |
| **Open Sans** | Friendly, readable | Newsletters, emails |
| **Lato** | Professional, neutral | Business proposals |
| **Montserrat** | Bold, impactful | Headlines, marketing |
| **Source Code Pro** | Code/technical data | Software documentation |
| **Crimson Pro** | Formal, traditional | Legal documents, books |
| **DM Sans** | Contemporary | Presentations |

#### How to Change the Font:

1. **Select the text** you want to change. (Click and drag your mouse over the text.)
2. Click the **Font dropdown** in the toolbar (it shows the current font name).
3. Choose your desired font from the list.

> **Example**: To make a report title look formal, select the title text and choose *Crimson Pro*.

#### Bonus: Smart Font Mapping

If you paste text from Microsoft Word or Google Docs that uses fonts like **Arial** or **Times New Roman**, the editor automatically converts them to the closest available font:
- Arial â†’ Roboto
- Times New Roman â†’ Crimson Pro
- Courier New â†’ Source Code Pro

This ensures your document looks consistent, even when combining content from different sources.

### 2.2 Adjusting Font Size

The editor offers **4 size options** to create visual hierarchy:

| Size | Appearance | Use Case |

## 3. Document Structure

This section covers how to organize your content with headings, lists, and special blocks.

---

### 3.1 Headings

Headings create a hierarchy in your document, making it easier to scan and navigate.

| Level | Size | Use Case |
|-------|------|----------|
| **Heading 1** | Largest | Document title |
| **Heading 2** | Very large | Major sections |
| **Heading 3** | Large | Sub-sections |
| **Heading 4** | Medium-large | Topics within sub-sections |
| **Heading 5** | Medium | Minor points |
| **Heading 6** | Smallest heading | Fine distinctions |

#### How to Apply a Heading:

1. Place your cursor on the line you want to make a heading (or select the text).
2. Click the **Header dropdown** in the toolbar (shows "Normal" by default).
3. Choose *Heading 1*, *Heading 2*, etc.

> **Example Document Structure**:
> ```
> Heading 1: Quarterly Report
>   Heading 2: Financial Summary
>     Heading 3: Revenue
>     Heading 3: Expenses
>   Heading 2: Operational Updates
> ```

#### Reverting to Normal Text:

If you accidentally apply a heading, select the text and choose **Normal** from the Header dropdown.

---

### 3.2 Text Alignment

Control how text is positioned horizontally.

| Alignment | Description | Use Case |
|-----------|-------------|----------|
| **Left** | Text starts from the left (default) | Body paragraphs |
| **Center** | Text is centered | Titles, quotes |
| **Right** | Text aligns to the right edge | Dates, signatures |
| **Justify** | Text stretches to fill both edges | Formal reports, newspapers |

#### How to Align Text:

1. **Select the paragraph(s)** you want to align.
2. Click the **Alignment** button in the toolbar (looks like horizontal lines).
3. Choose your alignment from the dropdown.

> **Example**: To center a document title:
> 1. Select the title text.
> 2. Click the Alignment dropdown.
> 3. Choose **Center**.

---

### 3.3 Text Direction (LTR/RTL)

If you're writing in a language that reads Right-to-Left (like Arabic or Hebrew), you can toggle the text direction.

#### How to Switch to RTL:

1. Place your cursor in the paragraph.
2. Click the **RTL** button in the toolbar (arrows pointing right-to-left).

The paragraph will now flow from right to left, and the alignment will flip accordingly.

## 4. Working with Tables

Tables are essential for organizing data in rows and columns. This editor includes a powerful table system.

---

### 4.1 Creating a Table

#### Steps:

1. Place your cursor where you want the table.
2. Click the **Table** button in the toolbar (grid icon).
3. A grid selector appearsâ€”hover to choose dimensions (e.g., 3 rows Ã— 4 columns).
4. Click to insert the table.

> **Example**: For a simple contact list, choose 3 columns (Name, Email, Phone) and as many rows as you have contacts.

---

### 4.2 Adding & Removing Rows/Columns

Once your table exists, you can modify its structure.

#### Adding Rows:

1. **Right-click** inside a cell.
2. From the context menu, choose:
   - *Insert Row Above* â€” adds a row above the current cell.
   - *Insert Row Below* â€” adds a row below the current cell.

#### Adding Columns:

1. **Right-click** inside a cell.
2. From the context menu, choose:
   - *Insert Column Left* â€” adds a column to the left.
   - *Insert Column Right* â€” adds a column to the right.

#### Deleting Rows/Columns:

1. **Right-click** inside the row or column you want to remove.
2. Choose *Delete Row* or *Delete Column*.

---

### 4.3 Merging & Splitting Cells

For complex layouts (like spanning headers), you can merge multiple cells into one.

#### Merging Cells:

1. **Click and drag** to select multiple adjacent cells.
2. **Right-click** on the selection.
3. Choose **Merge Cells**.

> **Example**: To create a title row that spans all columns:
> - Select the cells in the first row.
> - Merge them.
> - Type your table title.

#### Splitting Cells:

If you've merged cells and want to undo it:

1. **Right-click** on the merged cell.
2. Choose **Split Cells**.

---

### 4.4 Enabling a Header Row

A header row gives your table a professional look by styling the first row differently (bold text, shaded background).

#### Method 1: Right-Click Menu

1. **Right-click** anywhere in the table.
2. Look for the **Header Row** checkbox or toggle option.
3. Enable it.

#### Method 2: Keyboard Shortcut

1. Click inside the table.
2. Press `Ctrl + Shift + H` (Windows) or `Cmd + Shift + H` (Mac).

> **Visual Effect**: The first row will display with:
> - Bold text
> - A colored or shaded background
> - A thicker bottom border

---

### 4.5 Resizing Columns & Tables

#### Resizing Individual Columns:

1. **Hover** your mouse over the border between two column headers.
2. Your cursor will change to a **resize cursor** (â†”).
3. **Click and drag** left or right to adjust the column width.

#### Resizing the Entire Table:

1. **Hover** over the **bottom-right corner** of the table.
2. A small **resize handle** (diagonal arrow icon) will appear.
3. **Click and drag** to make the table wider or narrower.

> **Visual Feedback**: While dragging, a tooltip will display the table's width as a percentage (e.g., "75%").

## 5. Inserting Media

Enhance your documents with images, videos, and other media.

---

### 5.1 Adding Images

#### Method 1: From URL

1. Click the **Image** button in the toolbar.
2. A prompt appears asking for the image URL.
3. Paste the image link (e.g., `https://example.com/photo.jpg`).
4. Click **Insert**.

#### Method 2: Upload from Computer

1. Click the **Image** button.
2. Choose "Upload" (if available).
3. Select a file from your computer.
4. The image will be embedded directly in the document.

---

### 5.2 Resizing Images

Once an image is inserted, you can resize it.

#### Using Drag Handles:

1. **Click the image** to select it.
2. **8 small handles** appear around the edges (corners and sides).
3. **Drag any handle** to resize:
   - Corner handles maintain the aspect ratio (proportional).
   - Side handles can distort the image (use carefully).

#### Using Preset Sizes:

When you select an image, a **floating toolbar** appears with quick options:

| Button | Effect |
|--------|--------|
| **25%** | Sets image width to 25% of the editor width |
| **50%** | Sets image width to 50% of the editor width |
| **75%** | Sets image width to 75% of the editor width |
| **100%** | Makes the image full-width |
| **Reset** | Reverts to the image's original dimensions |

#### Deleting an Image:

1. Select the image.
2. Click the **Delete** button in the floating toolbar, OR press the **Delete** or **Backspace** key.

---

#### How to Align:

> **Example**: For a newsletter, you might float a product image to the right with a description wrapping on the left.

## 9. Tips & Best Practices

### Document Organization

- **Use Headings Consistently**: Create a clear hierarchy (H1 for title, H2 for sections, H3 for sub-sections).
- **One Idea Per Paragraph**: Keep paragraphs focused for easier reading.
- **Use Lists for Steps or Items**: Bullets and numbers are easier to scan than long paragraphs.

### Visual Design

- **Limit Font Variety**: Stick to 2-3 fonts maximum in one document.
- **Use Color Sparingly**: Too many colors can be distracting.
- **Align Images Thoughtfully**: Float images left/right only when text should wrap around them.

### Tables

- **Always Use Header Rows**: They make data tables much easier to understand.
- **Keep Tables Simple**: Avoid deeply nested or merged cells unless absolutely necessary.
- **Size Tables Appropriately**: A 75% width table often looks better than a 100% width one.

### Before Sharing

- **Proofread**: Use the zoom feature to review your document at 100% and 150%.
- **Check Links**: Click each link to ensure it goes to the correct destination.
- **Clear Unwanted Formatting**: If you pasted content from various sources, do a final formatting check.
